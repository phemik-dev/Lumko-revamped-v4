# LumkoMDX Revamped v4

Production-ready marketing site for LumkoMDX, built with Next.js, TypeScript, and Tailwind CSS.

## Structure

- `app/` - App Router pages and global layout.
- `components/` - Shared shell, section, visual, CTA, card, and form primitives.
- `content/` - Local structured site copy.
- `lib/` - Small shared utilities.

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
